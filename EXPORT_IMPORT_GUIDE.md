# PrivateVault 导出导入功能说明

## 📋 功能概览

PrivateVault 现在支持**双向操作**：
- **导入**：从系统相册复制文件到私密空间
- **导出**：从私密空间复制文件到共享区域

---

## 🔒 隐私保护机制

### 存储位置说明

```
应用数据目录/
├── files/                    # 应用私有目录
│   ├── manga/                # 漫画（私密，系统相册看不到）
│   │   └── .nomedia          # 防止扫描
│   ├── video/                # 视频（私密，系统相册看不到）
│   │   └── .nomedia          # 防止扫描
│   ├── photo/                # 图片（私密，系统相册看不到）
│   │   └── .nomedia          # 防止扫描
│   └── exports/              # 导出区（相对私密）
│       └── .nomedia          # 防止系统相册扫描
│
└── cache/                    # 缓存目录
```

### 三个区域的隐私级别

| 区域 | 系统相册可见 | 文件管理器可见 | USB 可见 | 用途 |
|------|------------|--------------|--------|------|
| `manga/video/photo` | ❌ 否 | ❌ 否 | ❌ 否 | 主存储区（最私密） |
| `exports` | ❌ 否 | ✅ 是 | ✅ 是 | 导出区（方便访问） |
| 系统相册（DCIM） | ✅ 是 | ✅ 是 | ✅ 是 | 公共区域 |

## 📥 导入功能

### 如何使用

1. 在 PrivateVault 中点击"添加照片/视频/漫画"
2. 系统弹出文件选择器
3. 选择一个或多个文件
4. 文件被**复制**到 `files/{manga,video,photo}/` 目录
5. 完成！现在只有 PrivateVault 能看到这些文件

### 重要提示

- ✅ 导入是**复制**操作，原文件保留在系统相册
- ✅ 如需要彻底私密，请手动删除系统相册中的原图
- ✅ 导入后文件受 .nomedia 保护，系统相册扫描不到

---

## 📤 导出功能

### 如何使用

1. 在 PrivateVault 中长按某个文件（或点击菜单按钮）
2. 在弹出菜单中选择"导出"
3. 文件被**复制**到 `files/exports/` 目录
4. 完成！

### 导出后的文件在哪里？

**路径：** `/Android/data/com.yourname.privatevault/files/exports/`

**访问方式：**

1. **使用文件管理器**
   - 打开手机自带的"文件管理"应用
   - 进入"内部存储" → "Android" → "data"
   - 找到 `com.yourname.privatevault` 文件夹
   - 进入 `files/exports/` 即可看到导出的文件

2. **通过 USB 连接电脑**
   - 用数据线连接手机和电脑
   - 在手机上选择"文件传输"模式
   - 电脑上打开手机存储
   - 导航到 `Android/data/com.yourname.privatevault/files/exports/`

### 导出后的隐私性

- ✅ **系统相册仍然看不到**：`exports` 目录也有 .nomedia 文件
- ✅ **文件管理器可以访问**：方便用户管理和分享
- ✅ **USB 连接可以传输**：方便备份到电脑

### 导出后想分享到微信/QQ？

1. 打开文件管理器
2. 找到导出的文件
3. 长按文件 → 点击"分享"
4. 选择微信/QQ/其他应用

## 🔁 典型使用场景

### 场景 1：下载私密照片

```
微信收到照片 → 保存到系统相册 → 导入到 PrivateVault → 删除系统相册原图
```

这样照片就完全私密了！

```
PrivateVault 私密照片 → 导出 → 从文件管理器分享 → 分享后可删除导出文件
```

既保护隐私，又方便分享！

### 场景 3：备份到电脑

```
PrivateVault 导出 → USB 连接电脑 → 复制 exports 目录到电脑 → 完成备份
```

## ⚠️ 注意事项

### 1. Android 11+ 限制

Android 11 及以上版本限制了访问 `Android/data` 目录：
- 部分文件管理器可能无法访问
- 推荐使用系统自带的"文件管理"应用
- 或使用"MT 管理器"等第三方工具

### 2. 卸载应用

- `files/` 目录的所有数据（包括 exports）都会被删除
- 导出重要文件前请先备份到电脑

### 3. 导出文件管理

当前版本导出文件会累积：
- 导出多次会生成多个文件
- 建议在文件管理器中定期清理不需要的导出文件

## 💡 使用技巧

### 技巧 1：批量导出

目前支持单个文件导出，建议：
- 选择需要导出的文件
- 一个一个导出
- 未来版本会支持批量导出

### 技巧 2：导出后重命名

如需分享但不想暴露原名：
1. 先在 PrivateVault 中重命名文件
2. 再导出
3. 这样导出的文件名就是新的

### 技巧 3：定期检查导出目录

建议每隔一段时间：
- 打开文件管理器
- 检查 `exports` 目录
- 删除不再需要的导出文件
- 保持导出区整洁

## ❓ 常见问题

### Q: 导出后系统相册能看到吗？
A: 不能！导出目录也有 .nomedia 文件，系统相册仍然看不到。

### Q: 导出和导入有什么区别？
A: 
- 导入：系统相册 → 私密区（完全隐藏）
- 导出：私密区 → 导出区（文件管理器可访问，但系统相册看不到）

### Q: 为什么不只用一个区域？
A: 分区设计更安全：
- 主存储区：完全私密，日常使用
- 导出区：相对私密，需要分享/备份时使用

### Q: 导出文件会被其他应用扫描到吗？
A: 不会，因为有 .nomedia 文件，微信、QQ 等应用的图片扫描功能也扫描不到。

### Q: 如何彻底删除导出的文件？
A: 有两种方式：
1. 使用文件管理器手动删除
2. 卸载 PrivateVault（会删除所有数据）
